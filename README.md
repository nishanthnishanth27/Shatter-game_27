# 🎮 Shatter Game

A fun and interactive web-based game built using HTML, CSS, and JavaScript. Challenge your reflexes and shatter the blocks!

## ✨ Features
- **Responsive Design:** Works on both mobile and desktop.
- **Smooth Animations:** High-quality shatter effects.
- **Score Tracking:** Real-time score updates as you play.

## 🕹️ How to Play
1. Open the [Live Link](https://nishanthnishanth27.github.io/Shatter-game_27/).
2. Use your mouse or touch to interact with the game elements.
3. Aim to shatter the objects and score high!
